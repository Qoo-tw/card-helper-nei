# 刷卡小幫手（女友版）

這是一個 **純前端 / 離線可用** 的小工具：輸入「店家/描述」與「金額」，就會在三張卡中自動推薦最划算的一張，並且可以一鍵記帳，累加本月的「已用/剩餘」回饋上限。

**只會用到三張卡：**
- 大戶黑卡
- 大衛 LinePay 卡
- 匯豐 Live+

> 重要：所有交易資料只存在你自己的手機/瀏覽器（localStorage），不會上傳到任何伺服器。

---

## 功能
- ✅ 店家 + 金額 → 自動推薦卡片與規則
- ✅ 顯示「本筆回饋估算」與「本月已用/剩餘」
- ✅ 一鍵新增交易紀錄（可刪除）
- ✅ 一鍵清空本月紀錄
- ✅ 離線可用（PWA + Service Worker）

---

## 規則資料在哪裡？
資料都放在 `data/` 目錄：

- `data/rules.json`：規則（回饋%、上限、是否需要 Line Pay、Live+ 是否限定分類）
- `data/merchantmap.json`：店家關鍵字 → 指定規則（例如：悠遊卡 → 悠遊卡3%）
- `data/categorymap.json`：店家關鍵字 → 推測分類（餐飲/購物/娛樂），給 Live+ 的 3% 規則用

你可以隨時新增關鍵字，不需要改程式。

---

## iPhone 安裝成 App（PWA）
1. 用 **Safari** 打開你的 GitHub Pages 網址（例如：`https://<你的帳號>.github.io/<repo>/`）
2. 點「分享」→「加入主畫面」
3. 之後就像 App 一樣可以離線打開使用

---

## GitHub Pages 部署
- Settings → Pages
- Source: Deploy from a branch
- Branch: `main` / `(root)`
- Save
- 等幾分鐘後就會有網址

---

## 注意
- Live+ 的「三大通路 3%」需要分類（餐飲/購物/娛樂）。你可以用「自動」讓系統猜，不準再改成手動。
- 若金額超過某規則「本月剩餘可刷」，畫面會提示「超出部分回饋可能降低」。
